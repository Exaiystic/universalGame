Welcome to my game!

Introduction

Military Clicker is a spin-off version of the popular game "Cookie Clicker", the game features similar elements to Cookie Clicker, as well as many other games within the idle genre, however, this game has a twist of that you are getting a bigger army with each click.

The aim of the game is to build the biggest army known to mankind. You can achieve this by training more and more soldiers, invading countries and buying better equipment for your soldiers to train with. If you would like to find out more about Military Clicker, you can visit the 'How to play' page.
